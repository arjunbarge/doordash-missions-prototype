# Known Limitations

This is an MVE (Minimum Viable Experience) interactive prototype designed strictly for a live pitch presentation.

- **No Real Backend:** All data is managed in client-side state using Zustand and mock catalogs.
- **No Real Payments:** Checkout flows are simulated.
- **No Live DoorDash Integration:** This does not connect to the actual DoorDash merchant APIs.
- **Prototype Scope Only:** Features outside the "Saturday Dinner Party" anchor mission are present as UI stubs only.
