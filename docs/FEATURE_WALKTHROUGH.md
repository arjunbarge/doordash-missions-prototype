# Feature Walkthrough

1. **Mission-First Home:** Users declare their intent ("Host a dinner party").
2. **Mission Templates:** Three curated styles are presented. User selects "Intimate Dinner for 6".
3. **Smart Mission Cart:** Items from 4 merchants are aggregated. AI provides "Why these picks?" reasoning.
4. **Mission-Aware Substitutions:** Tapping "Try a substitution" on the wine offers a Sancerre or Vouvray swap based on the mission context.
5. **Mission + Item Feedback:** A post-event flow captures thumbs up/down ratings and notes.
6. **Mission Memory:** A timeline of past missions allowing users to run successful events again.
